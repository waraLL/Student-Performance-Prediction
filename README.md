Student Performance Prediction

Este proyecto tiene como objetivo predecir el rendimiento académico de estudiantes utilizando técnicas de Machine Learning. A partir de variables relacionadas con hábitos de estudio, asistencia y desempeño previo, se busca identificar si un estudiante aprobará o no.
Se implementaron dos modelos de clasificación: 
- Regresión Logística
- Random Forest
Evaluando su desempeño mediante métricas como accuracy, precision, recall y F1-score. Además, se desarrolló un sistema de interpretación basado en probabilidades que permite clasificar a los estudiantes en niveles de riesgo académico (bajo, medio y alto), incorporando factores explicativos para facilitar la toma de decisiones educativas.

Este enfoque convierte el modelo predictivo en una herramienta de apoyo para la identificación temprana de estudiantes que requieren intervención académica.
